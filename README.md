# todomvc-app
##开始使用
    -安装npm、git
    -git clone https://github.com/guoshiqiufeng/todomvc-app.git
    -使用终端进入项目所在目录
    -npm install
##页面介绍
    -index.html使用普通实现方式实现
    -index2.html使用路由的方式实现
    -indexV3.html使用模块分离+路由实现+本地存储
